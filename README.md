sense-js
========

html5 sensor interaction library for mobile

Demo site!

Set up the virtualenv in the directory, by doing:

'''sh
  virtualenv venv
  source venv/bin/activate
'''

Install the requirements (once)

'''sh
  virtualenv venv
  (venv) pip install -r requirements.txt
'''

Start the application:
'''sh
  (venv) python app.py
'''
